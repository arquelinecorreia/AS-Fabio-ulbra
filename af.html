Relação entre dados e Hardware são juntas pois sem dados o hardware não irá saber como calcular, medir ou fazer ações pois são com esses dados, que o hardware saberá como uar o HD, como carregar videos em sites, como forçar o processador ao maximo ou a minimo da força.

A Relação entre dados e Software é que com dados o software pode calibrar especificamente os dados, como também é necessário para que os softwares adaptem aos dados fazendo com que não tenha bugs pois, os dados farão o software ser calibrado.
